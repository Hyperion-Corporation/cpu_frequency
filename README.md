# CPU Information
CPU Informaion on Windows, OSX and Linux
# Requirements
On Windows: No, if you use .exe file.

On Linux: Python3 and figlet
# HOW TO USE
```python3
python cpu_info.py
```
or (only Linux)
```python3
chmod +x cpu_info.py
./cpu_info.py
```
If you want to import the cpu_information:
```python3
import cpu_info
```
